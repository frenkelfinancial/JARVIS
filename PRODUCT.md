# Product

## Register

product

## Users

Jace — solo operator, financial/insurance entrepreneur. Uses this daily as a personal command center to monitor and orchestrate AI agents running autonomously.

## Product Purpose

JARVIS is a personal AI agent ecosystem dashboard. It shows live status of five autonomous bots (commission tracking, lead monitoring, video generation, ecommerce, and the JARVIS orchestrator). Success = Jace glances at the screen and instantly knows what each agent is doing.

## Brand Personality

Pixel command center. Retro-technical. Sovereign.

## Anti-references

- Generic SaaS dashboards (Notion, Linear, Vercel) — too clean, no character
- Flashy glassmorphism / gradient-heavy cyberpunk — decorative noise over function
- Top-down God-mode enterprise tools — impersonal

## Design Principles

1. **Glanceable first** — status, progress, and agent names readable in under 2 seconds
2. **Pixel aesthetic is intentional** — the retro game feel is the brand, not decoration
3. **Personal scale** — one user, five agents; no need for multi-user abstractions
4. **Names are yours** — every character can be renamed; it's your empire

## Accessibility & Inclusion

Internal tool, single user. Reasonable contrast on dark bg. Respects `prefers-reduced-motion`.
